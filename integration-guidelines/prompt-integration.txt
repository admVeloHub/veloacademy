PROMPT PARA INTEGRAÇÃO DE LAYOUT - VELOACADEMY

Olá! Você está trabalhando em uma página que será integrada ao projeto VeloAcademy. Para garantir consistência visual e funcional, siga estas instruções:

1. LEIA O ARQUIVO README.md nesta pasta para entender os padrões de design

2. APLIQUE AS SEGUINTES ALTERAÇÕES NA SUA PÁGINA:

   a) ESTRUTURA HTML:
   - Use a estrutura base fornecida no README.md
   - Inclua o header padrão com navegação completa
   - Mantenha o atributo data-theme="light" no html
   - Use o título no formato "[Nome da Página] - VeloAcademy"

   b) CSS:
   - Remova todos os estilos customizados que não sigam o padrão VeloAcademy
   - Use APENAS as variáveis CSS definidas no projeto
   - Aplique as classes .btn, .btn-primary, .btn-secondary para botões
   - Use .dashboard-card para cards/seções
   - Mantenha a fonte Poppins

   c) NAVEGAÇÃO:
   - Inclua TODAS as abas: Home, Cursos, Conquistas, Auto Feedback, VeloHub
   - Marque a aba atual como "active"
   - Mantenha os links corretos para cada página

   d) FUNCIONALIDADES:
   - Inclua o script veloacademy.js
   - Mantenha o toggle de tema funcionando
   - Preserve a responsividade

3. VERIFICAÇÕES OBRIGATÓRIAS:
   - A página deve funcionar com tema claro e escuro
   - Deve ser responsiva em mobile e desktop
   - Deve manter consistência visual com index.html e cursos.html
   - Todos os links de navegação devem funcionar
